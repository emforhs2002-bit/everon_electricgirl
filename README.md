# 에버온 전기요금 검증 시스템 V2

에버온과 한전 청구서를 교차검증하는 웹 도구. 매월 15,000개 고객번호의 청구서 오류·누락·과소청구를 자동으로 찾아냅니다.

## 주요 기능

### 단일 HTML 파일로 동작
- 서버 없음. 브라우저에서 바로 실행
- 모든 데이터는 브라우저 내 IndexedDB에만 저장 (외부 전송 없음)
- GitHub Pages에 올리면 어디서나 접속 가능

### 29개 자동 검증 규칙

**금액 검증 (A1~A8)**
- 시간대별 사용량 합산, 기본요금 단가 계산, 기본요금 상한, 초과요금, 공급가액, 부가세, 전력기금, 청구요금 합산

**데이터 이상 (A9~A12)**
- 사용량 있는데 시간대별 0값, 미납요금, 검침기간 비정상, 전월지침 이상

**교차검증 (B12~B14)**
- 관제 vs 한전 계약전력, 시간대별 사용량, 총 사용량 (관제>한전은 과소청구 의심)

**계약 적정성 (C16~C17)**
- 충전기 정격용량 합 vs 계약전력 (**과다계약 즉시 감지**)
- 최대수요전력 / 계약전력 비율 (과다·부족 계약)

**월간 변동 (D18~D20)**
- 사용량 ±50% 급변, 청구액 ±50% 급변, 요금적용전력 상승 (피크월 잠금 경고)

**중복·누락·신규 (E21, F22~F24)**
- 같은 고객번호 중복청구, 한전 누락 의심, 2개월 연속 누락 = 해지, 신규 충전소/증설 감지

### 대시보드
- 월별 청구액 트렌드
- 충전기 가동률 분포
- 최근 주요 이슈 Top 5

### 1년치 배치 업로드
- 통합 CSV/XLSX 파일 한 번에 업로드 → 청구년월별 자동 분리 저장
- 월별로 검증 실행 버튼 (대시보드에서 바로 선택)

### 엑셀 내보내기
- 원본 엑셀에 색칠 (치명/주의/참고) + 셀 코멘트
- Claude 계산값 열 추가 (Claude_기본요금, Claude_공급가액 등)
- 이상 플래그 열 (시간대 0값, 전월지침 0, 미납, 중복청구)
- 검증요약 시트 + 검증정보 메타 시트

## 사용법

### 1. 접속 & 로그인
- GitHub Pages 주소 접속
- 아이디: `everon` / 비번: `somin`

### 2. 초기 배치 업로드 (1년치)
- 상단 우측 `📦 1년치 배치 업로드` 버튼
- 빌코리아 통합 CSV + 관제 통합 CSV 각각 올리기
- 청구년월/요금년월별로 자동 분리 저장

### 3. 월별 검증 실행
- 대시보드의 `📅 누적 월별 데이터` 테이블에서 검증하고 싶은 월의 `검증 →` 버튼 클릭
- 해당 월의 빌코리아 × 관제(전월) 자동 매칭해서 29개 규칙 전체 실행

### 4. 결과 확인
- 좌측 사이드바에서 카테고리별 이슈 확인 (뱃지로 건수 표시)
- 각 카테고리에서 `📥 엑셀 다운로드` 버튼으로 결과 엑셀 받기

### 5. 설정 조정
- 상단 ⚙️ 버튼
- 단가 (기후환경요금, 연료비, 부가세율 등), 오차 허용, 시간대 매핑, 해지 기준 등 조정

### 6. 이력 관리
- 상단 📚 버튼
- 누적된 월별 데이터 조회 · 삭제
- 백업 JSON 다운로드 (다른 브라우저/PC로 이동 시)
